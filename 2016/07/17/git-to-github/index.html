用Git把本地代码传到Github上

发表于 2016-07-17 | 分类于 技术 | | 阅读次数

建立git仓库

　　cd到你的本地项目根目录下，执行git命令，使用当前目录作为git仓库

　　此命令会在当前目录下创建一个.git目录。

跟踪与提交

　　如果当前你目录下有几个文件想要纳入版本控制，需要先用git add命令告诉Git开始对这些文件进行跟踪，然后提交

在github上创建仓库，作为远程仓库

　　在github上创建一个新的仓库，点击New Repository,点击Create repository，创建完成后拿到仓库的https地址，如https://github.com/chenweilu119/datatable.git

将本地仓库关联到远程仓库github上

上传代码到github远程仓库

　　执行完后，如果没有异常，等待执行完就上传成功了，中间可能会让你输入Username和Password，你只要输入github的账号和密码就行了。到这步为止，就上传成功了。打开github就可以看到在master分支下有从本地仓库上上传的文件了。
　　如果需要在github上展示一些demo，自己做的一些网页等，需要用到gh-pages分支。

创建gh-pages分支

gh-pages与master内容同步

访问页面

　　输入https://用户名.github.io/仓库名/ 就可以访问页面啦，如
https://chenweilu119.github.io/datatable/。!!! 注意要访问的页面的html文件名字必须是index.html
